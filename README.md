# AppStore Playground
===

AppStore Playground is a sample for using itunes rss api in Swift

## Features

* Using MVVM as basic architecture
* Using Reactive Programming
* Enable offline cache


## Getting Started

Run the project

```
pod install
```

Then open AppStorePlayground.xcworkspace and compile it
